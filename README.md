Job Application Tracker

This is a Python Tkinter app to track your job applications.

Features
- User login and signup
- Add new job applications
- Update application status and notes
- View a dashboard with totals and recent updates
- Export and view applications in a report
Files
- `widget_Portion.py` — the main Python code
- `data.csv` — your application data
- `history.csv` — call or action history
